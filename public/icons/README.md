# Icons untuk PWA

Direktori ini berisi icon untuk Progressive Web App (PWA).

## Icon yang Diperlukan

1. **icon-192.png** - 192x192 pixels
2. **icon-512.png** - 512x512 pixels

## Cara Membuat Icon

### Opsi 1: Menggunakan Tool Online
1. Kunjungi https://realfavicongenerator.net/ atau https://www.pwabuilder.com/imageGenerator
2. Upload logo/ikon SampahKuPilah
3. Generate icon dengan ukuran 192x192 dan 512x512
4. Download dan simpan di direktori ini

### Opsi 2: Menggunakan Image Editor
1. Buat gambar dengan ukuran 192x192 dan 512x512 pixels
2. Gunakan warna hijau (#4caf50) sebagai tema
3. Simpan sebagai PNG dengan nama:
   - `icon-192.png`
   - `icon-512.png`

### Opsi 3: Menggunakan Icon Generator (icon-generator.html)
Buka file `icon-generator.html` di browser untuk membuat icon sederhana.

## Catatan
- Icon harus berbentuk persegi (square)
- Format: PNG dengan transparansi
- Warna tema: #4caf50 (hijau)
